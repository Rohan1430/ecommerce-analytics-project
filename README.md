# 🛒 E-Commerce Analytics Dashboard

## 📌 Project Overview
This project is a full-stack data analytics system built using **Python, MySQL, and Streamlit**.  
It analyzes e-commerce data and provides business insights through an interactive dashboard.

---

## 🚀 Features
- 🔥 Top Selling Products Analysis
- 💰 Highest Spending Users
- 📈 Revenue by Category
- ⭐ Best Rated Products
- 📦 Order Details (Multi-table JOIN)
- 📊 Interactive Dashboard using Streamlit

---

## 🧱 Tech Stack
- **Python**
- **MySQL**
- **Streamlit**
- **Pandas**

---

## 🗄️ Database Design
The system uses multiple related tables:
- Users
- Products
- Categories
- Orders
- Order Items
- Reviews
- Payments
- Shipping
- Cart & Wishlist

---

## 🔗 Key Concepts Used
- SQL Joins (INNER JOIN, LEFT JOIN)
- GROUP BY & Aggregation
- Subqueries
- Database Relationships (Foreign Keys)
- Data Visualization
